# Flash — 设计规范文档

> 版本：v1.0 | 更新日期：2026-06-18

---

## 一、设计原则

- **极简**：每一个元素都要问"能不能没有它"
- **快速**：视觉层次清晰，一眼读完
- **安静**：不争夺注意力，淡蓝色传递冷静感

---

## 二、色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色（淡蓝） | `#87CEEB` | 按钮、强调元素 |
| 主色浅底 | `#E3F2FD` | 卡片背景、按钮涟漪 |
| 主色深 | `#5C9FBF` | 按下状态、文字链接 |
| 背景白 | `#FFFFFF` | 页面背景 |
| 背景灰 | `#F5F5F5` | 列表卡片背景 |
| 主文字 | `#1A1A1A` | 念头内容 |
| 次要文字 | `#9E9E9E` | 倒计时、辅助信息 |
| 删除红 | `#FF6B6B` | 左滑删除、过期警告 |
| 分割线 | `#EEEEEE` | 列表分隔 |

## 七、交互动效

### 按住录音
- 按钮从 scale(1) → scale(1.1)，过渡 0.15s
- 波纹动画：从按钮中心向外扩散的淡蓝圆环

### 录音完成
- 按钮弹回 scale(1)，过渡 0.2s，弹性曲线
- 轻微震动反馈（haptic feedback）

### 左滑删除
- 左滑露出红色删除区域
- 点击删除或继续左滑到底触发删除
- 删除时卡片向右滑出，下方卡片上移

### 自动消失
- 无动画，直接消失（安静原则）
