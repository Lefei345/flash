# Flash — 产品需求文档

> 版本：v1.0 | 更新日期：2026-06-18

---

## 一、产品定位

**Flash** 是一个极简念头捕捉器。

核心理念：**不给大脑留负担。** 想到什么说什么，重要的自然会处理，不重要的自动消失。

目标用户：任何被碎片想法困扰的人——灵感、待办、提醒、情绪记录。

---

## 二、核心功能

### 功能 1：语音丢念头（核心交互）

| 项目 | 说明 |
|------|------|
| 触发方式 | 打开 App，看到大按钮，**按住说话** |
| 结束方式 | **松手即存** |
| 存储形式 | 语音自动转文字后保存（不保留原始音频） |
| 时间限制 | 无最长录音限制，但鼓励短念头 |
| 操作步骤 | 1 步（按住→说话→松手） |
| 耗时目标 | ≤ 5 秒 |
| 无需操作 | 不用打字、不用分类、不用设时间、不用起标题 |

### 功能 2：念头流列表

| 项目 | 说明 |
|------|------|
| 排列方式 | 按时间倒序（最新在上） |
| 每条显示 | 文字内容 + 倒计时（"X 小时 Y 分钟后自动消失"） |
| 不显示 | 没有勾选框、没有优先级标签、没有分类、没有已读/未读 |
| 交互 | 点击无操作（纯浏览）；支持左滑手动删除 |

### 功能 3：自动过期删除

| 项目 | 说明 |
|------|------|
| 默认存活 | 24 小时 |
| 可选时间 | 用户可选择 1h / 6h / 24h / 3天 |
| 过期行为 | 自动删除，无声无息，无通知 |
| 设计意图 | 不重要的自然消失；重要的你会在消失前处理 |

---

## 三、设计约束

- 主色调：淡蓝色系
- 风格：极简、干净、留白多
- 页面数：2 个（录音页 + 列表页）
- 无账户系统、无登录、无注册

---

## 四、数据与隐私

- 所有数据仅存储在手机本地
- 不上传任何服务器
- 无第三方统计/追踪
- 离线完全可用

---

## 五、不做的事情（范围边界）

- ❌ 没有分类/标签/文件夹
- ❌ 没有优先级/星标
- ❌ 没有搜索（念头自己会消失，不需要搜）
- ❌ 没有云同步
- ❌ 没有导出功能
- ❌ 没有分享功能
- ❌ 没有编辑功能（说完就定了，想改就删了重新说）
- ❌ 没有多设备同步
- ❌ 没有通知推送
- ❌ 没有小组件

---

## 六、用户故事

1. 我在走路时突然想到一个点子 → 掏出手机 → 打开 Flash → 按住说话 → 松手 → 放回口袋。3 秒完成。
2. 睡前刷列表，看到"明天寄快递"只剩 2 小时了 → 决定明天一早就办 → 安心睡觉。
3. 过了一周回看，列表永远不超过 10 条——大部分念头都在 24 小时内自动消失了。
